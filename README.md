# Trackalendar

Universāls lifestyle tracking kalendārs — dark theme, klikšķināmi datumi, kategoriju sistēma ar emoji un krāsām.

## Lietošana

1. Atver `index.html` pārlūkā vai caur GitHub Pages: `magnificolv.github.io/trackalendar/`
2. Klikšķini uz datuma → atveras modāls ar dienas ierakstiem
3. Pievieno ierakstus: izvēlies kategoriju, ieraksti nosaukumu, laikus, piezīmes
4. Pārvaldi kategorijas caur "+ Pievienot" pogu leģendā

## Tehniskā informācija

- Viens `index.html` fails — nav atkarību, nav servera
- Dati glabājas localStorage (`trackalendar-v1`)
- GitHub Pages compatible
